# 算法学习网页 — 设计规格

## 概述

一个纯 HTML/CSS/JS 算法学习平台，深色编辑室美学，支持可视化动画、分步讲解、代码并排三种交互模式。MVP 覆盖 4 种排序算法 + 二分查找。

## 技术栈

- 纯 HTML/CSS/JS，零依赖，零构建工具
- Canvas API 绘制动画
- Google Fonts CDN 加载字体

## 美学方向：暗黑编辑室 (Dark Editorial)

- 基调：深色画布上极简排版，单一暖橙色为信号色
- 字体：标题 DM Serif Display，正文 DM Sans，代码 JetBrains Mono
- 微交互：淡入位移过渡、hover 箭头滑入、动画运行时呼吸光晕

## 架构

### 布局

```
┌──────────┬──────────────────────────────────────────┐
│ 侧边栏    │ 主内容区                                  │
│ 240px    │  顶部：算法标题 + 模式切换                   │
│          │  中部：可视化/代码/分步面板                   │
│ 算法导航  │  底部：控制栏                              │
│ + 设置   │                                          │
└──────────┴──────────────────────────────────────────┘
```

### 文件结构

```
index.html
css/
  base.css          — 变量、重置、字体引入
  layout.css        — 侧边栏 + 主内容区网格
  sidebar.css       — 导航样式
  controls.css      — 按钮、滑块、控制栏
  visualizer.css    — Canvas 容器 + 动画光环
  code-panel.css    — 代码块样式 + 语法高亮
  step-panel.css    — 分步讲解面板
js/
  app.js            — 入口，绑定事件，切换模式/算法
  sidebar.js        — 侧边栏导航逻辑
  renderer.js       — Canvas 绘制引擎
  algorithms.js     — 算法实现（纯数据，不涉及 UI）
  controls.js       — 播放/暂停/步进/速度控制
  code-display.js   — 代码注入 + 行高亮
  step-display.js   — 分步模式的数据状态展示
```

### 数据流

App 持有当前 `{ algorithm, mode }` 状态。controls.js 发出播放/暂停/步进事件。renderer.js 每帧从 algorithms.js 获取步骤数据，绘制到 Canvas。

## 三种交互模式

### 模式 1：可视化动画
- Canvas 全宽，requestAnimationFrame 驱动
- 控制栏：播放/暂停/上一步/下一步/重置
- 速度滑块 0.5x ~ 3x
- 底部当前操作说明文字

### 模式 2：分步讲解
- 左侧 70% Canvas，用户点"下一步"前进一帧
- 右侧 30% 当前代码行高亮 + 数据状态快照
- 每步附文字说明

### 模式 3：代码 + 可视化并排
- 左侧 50% 完整源码 + 行高亮
- 右侧 50% 同步 Canvas 动画
- 纯 CSS 语法高亮

## MVP 算法清单

| 算法 | 可视化方式 | 难度 |
|------|-----------|------|
| 冒泡排序 | 柱状图，相邻比较高亮闪烁 | 入门 |
| 选择排序 | 柱状图，橙色扫描线 + 交换 | 入门 |
| 插入排序 | 柱状图，已排序区间绿色标记 | 入门 |
| 快速排序 | 柱状图，基准高亮，分区扫描 | 进阶 |
| 二分查找 | 横向数字卡片，搜索区间收缩 | 入门 |

数据规模默认 20，范围 5-50。二分查找默认 15 个有序元素。

## 边界处理

- 切换数据量 → 停止动画，重置数据集
- 切换模式 → 保留进度
- 动画进行中 → 算法切换按钮置灰
- 极端数据量 → Canvas 自适应柱宽/间距
- 二分查找目标值 → 标题栏显示，可点选

## 非功能需求

- Canvas 按需渲染，无动画时不运行 rAF 循环
- 首次加载 < 200KB（含字体 CDN）
- 支持 Chrome/Edge/Firefox 最新两版
- 无 polyfill，无第三方库

## 不包含

- 用户账号/进度存储
- 响应用户自定义输入数据
- 移动端适配（面向桌面端）
- 音频/音效
